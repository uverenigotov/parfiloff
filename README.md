# parfiloff
Александр Парфилов — автор проекта «Уверен и Готов», практикующий инженер и владелец мастерской электроники «Армавиртелеремонт».

20 лет опыта практического телеремонта на верстаке.

Цель проекта — показать, как скрестить многолетний практический опыт работы руками с возможностями современных нейросетей для автоматизации бизнес-процессов.
